Registration

This section summarizes how W&M registration works for transfer students, and points you to the step-by-step “How to Register” and the “PATH 101” primer.

Start here

How to Register →   PATH 101 →   Open PATH →

Dates/times vary by term; see the Registrar’s “When to Register” and New Students timelines for the latest schedule.

Key ideas

  • Before your time ticket opens, you build a Primary Cart in PATH and set registration priorities (1st, 2nd, …) and alternatives.
  • Co-requisites (e.g., lecture + lab) are processed together at the highest priority in the group.
  • Holds and unresolved registration errors can block cart processing—clear them before carts close.

See demos and the Registrar’s Quick Start PDF linked in PATH 101.
